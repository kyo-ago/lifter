# lifter

Local Proxy for frontend developers.

### Usage

Launch by `yarn run app`.

    yarn install
    yarn run app

## Development

    yarn install
    yarn run bootstrap

## Running tests

Install devDependencies and Run `npm test`:

    npm i -d && npm test
